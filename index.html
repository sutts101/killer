Welcome to Ian's Killer Sudoku

The objective is to fill the grid with numbers from 1 to 9 such that:

  • Each row, column, and "nonet" (3x3 box of cells) contains each number exactly once
  • The sum of all numbers in a "cage" (region surrounded by dotted lines) matches the number shown in its corner
  • No number appears more than once in a cage

To play:

  • Use the mouse and/or cursor keys to focus on a cell
  • Use the keyboard to enter possible values (for example, if you have a cage with 2 cells and a sum of 3, go to each cell and enter 1 followed by 2)
  • Every time you enter a value, it will add it to the cell if is not already there, else remove
  • To clear all values from a cell, press c
  • To undo previous entry press z
  • To redo previous entry press y

About this Killer Sudoku

Written in CoffeeScript with copious help from Mocha + should.js. Uses HTML5 Canvas / Local Storage for render and persistence.

My first goal was to get it to the point where it works and is at least as good any other player I've ever seen. I think I'm well on the way to achieving that - please let me know if you think differently!!

A really interesting challenge with Sudokus in general is to write a Solver that thinks like a human (as opposed to just applying computational brute force). That could open the door to some features to help (really help!) a user when they're stuck. This is something I'm hoping to have a play with a little further down the road...
